> Részletes vásárlási flow az e-commerce best practice-ek alapján

---

## Kutatási források

A flow tervezésnél az alábbi forrásokat vettem figyelembe:
- [Shopify Checkout Flow](https://www.shopify.com/blog/ecommerce-checkout-flow) - Checkout optimalizáció
- [Bolt Checkout Design](https://www.bolt.com/thinkshop/ecommerce-checkout-process-flow) - Checkout UX
- [BigCommerce Cart Abandonment](https://www.bigcommerce.com/articles/ecommerce/abandoned-carts/) - Kosár elhagyás csökkentése
- [ConvertCart Best Practices](https://www.convertcart.com/blog/checkout-page-examples) - Checkout design példák
- [GotPhoto](https://www.gotphoto.com/school/) - Iskolai fotó rendelés specifikus

## Statisztikák (iparági átlag)

| Metrika | Érték | Forrás |
|---------|-------|--------|
| Kosár elhagyási ráta | 70.19% | Baymard Institute |
| Mobile kosár elhagyás | 85%+ | Salesforce |
| Éves bevétel veszteség | $18B | E-commerce összesen |
| Konverzió növekedés 5+ review-val | +270% | Spiegel Research |

**Cél**: 70% → 50% kosár elhagyás = jelentős bevétel növekedés!

## Belépési pontok

### A) Bejelentkezett user

```
/albums → Album kiválasztás → /photos/:albumId
```

### B) Guest link (email/SMS)

```

│                           EMAIL / SMS                                       │
```

## Analitika események

| Esemény | Trigger | Adat |
|---------|---------|------|
| `gallery_viewed` | Gallery page load | album_id, photo_count |
| `photo_added_to_cart` | Add to cart | photo_id, size, qty, price |
| `cart_viewed` | Cart page load | item_count, total |
| `checkout_started` | Checkout auth load | cart_value |
| `checkout_step_completed` | Step navigation | step_name |
| `coupon_applied` | Coupon success | code, discount |
| `shipping_selected` | Shipping choice | method, cost |
| `payment_initiated` | Pay button click | method, total |
| `order_completed` | Success page | order_id, total, items |
| `cart_abandoned` | Exit + 1hr no return | cart_value, items |
